# Landing-Page

## Mobile first

## Install styled-components

1. yarn add styled-components
2. In package.json :

```json
{
  "resolutions": {
    "styled-components": "^5"
  }
}
```

## Install material-ui

1. yarn add @material-ui/core @material-ui/icons

## Install eslint airbnb

create .env EXTEND_ESLINT=true
